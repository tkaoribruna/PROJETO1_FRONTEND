# 🌹REPOSITÓRIO DO PROJETO DA DISCIPLINA WEB-FRONT-END 🌹
---

O projeto, desenvolvido no primeiro semestre de 2025, tem como objetivo a criação de uma interface web para uma floricultura, apresentando produtos de forma visualmente atrativa e proporcionando uma navegação simples e intuitiva ao usuário.
